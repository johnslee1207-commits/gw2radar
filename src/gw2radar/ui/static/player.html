Trust center

Privacy & Safety

Private data boundaries

  • API keys are submitted to the backend and never rendered back to this page.
  • Reports use summaries and evidence references, not raw private API responses.
  • Public KB, patch, market, and build evidence stays separate from account snapshots.

Player safety boundaries

  • No automatic gameplay inputs.
  • No game memory reading or game file modification.
  • No automatic buy, sell, listing, or order placement.
  • No crafting or chat automation.
  • No guaranteed profit or absolute meta claims.
