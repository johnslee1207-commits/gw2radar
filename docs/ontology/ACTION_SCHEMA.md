# Action Schema

Required action fields:

- id
- action_type
- title
- description
- target_entity_id
- target_goal_id
- preconditions
- expected_outputs
- costs
- constraints
- priority_score
- urgency
- reason_codes
- evidence_refs
- properties
- explanation
- created_at

MVP 0.1 action types are manual recommendations only. No automatic gameplay, trading, client control, or order execution is allowed.
