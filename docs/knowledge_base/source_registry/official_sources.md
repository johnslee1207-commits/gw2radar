# Official Sources

This file documents the intended source registry seed for official GW2 sources.

Planned source entries:

| Name | Type | Allowed Use | Crawl Policy | Rate Limit Policy |
|---|---|---|---|---|
| Official GW2 API | official_api | api_json | api_only | gateway_managed |
| Official GW2 Wiki | official_wiki | summary_and_reference | conservative | low_frequency |
| Official GW2 announcements | official_news | summary_and_reference | manual_only | manual |

No credentials, private account payloads, or copied full text belong in source registry documents.
