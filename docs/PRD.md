# Product Requirements Document

## Product Directions

1. AI Export Website Kit Generator
2. AI Technical Proposal / PDF Report Generator

## Shared Promise

Collect structured input and generate professional deliverables that can be given to customers, managers, developers, website agencies, or AI coding tools.

## Export Website Kit Deliverables

- Website Strategy Report
- Website Page Copy
- SEO Keyword Map
- CMS Content Model
- Developer Task List
- Technical Implementation Plan

## Technical Proposal Deliverables

- Background
- Goals and Constraints
- Architecture
- Module Design
- Roadmap
- Budget Estimate
- Risks
- Acceptance Criteria

## MVP Boundary

MVP is a paid deliverable workflow, not a full SaaS platform. It should support intake, generation, validation, export, and delivery package creation.

## First Vertical Slice

Use sample JSON inputs, deterministic MockGenerationProvider, validation, Markdown/CSV export, and smoke harness.
